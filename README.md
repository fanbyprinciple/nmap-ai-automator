# NMAP Automator with gemini

How to use it

API Key: Get one from the Google AI Studio.

>https://aistudio.google.com/api-keys

Export your API Key in your terminal session:
  

> export GEMINI_API_KEY="your_actual_api_key_here"

Run the script with the new type:
Bash

> ./nmap_automator.sh -H 192.168.1.10 -t Analyze

Otherwise it works as nmap automator script by @21y4d

https://github.com/21y4d/nmapAutomator
